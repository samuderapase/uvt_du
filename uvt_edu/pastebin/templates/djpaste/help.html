DjPaste is a pastebin written in Django. It was built as part of 7days7apps project. It is heavily stolen inspired from Dpaste.com, but the codebase is in no way related.

Pasting content with DjPaste is very easy. You just choose the type of the code you are pasing, paste, and hit the button. Magic starts happening. Only text is the required field. All other fields are optional.

Super Special Features:

  • Automatically focuses to Code field, so you can paste fast.
  • Remembers previous choices.
  • No unnecessary fields.
